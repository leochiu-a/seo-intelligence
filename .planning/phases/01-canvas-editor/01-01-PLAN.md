**Step 1: Create src/components/UrlNode.tsx**

This is a custom React Flow node registered via `nodeTypes`. It receives `NodeProps` from reactflow.

Structure:
```tsx
import { memo, useState, useCallback } from 'react';
import { Handle, Position, type NodeProps } from 'reactflow';
import { Pencil } from 'lucide-react';
import { EditPopover } from './EditPopover';
```

Node data type (define in same file or inline):
```tsx
export interface UrlNodeData {
  urlTemplate: string;
  pageCount: number;
}
```

The component renders a card div with these exact classes:
- Outer wrapper: `group relative min-w-[200px] max-w-[280px] min-h-[64px] bg-white border border-gray-200 rounded-lg shadow-sm hover:shadow-md border-l-4 border-l-indigo-500 px-4 py-2`
- URL template text: `text-sm font-semibold text-gray-900 leading-tight truncate` (14px / semibold per UI-SPEC)
- Page count text: `text-xs text-gray-500 leading-snug mt-1` with format: `{N} pages` (singular `1 page` when pageCount === 1)
- Pencil button: `absolute top-2 right-2 opacity-0 group-hover:opacity-100 transition-opacity p-1.5 rounded hover:bg-gray-100` with `aria-label="Edit node"`
- Pencil icon: `<Pencil size={14} className="text-gray-400 hover:text-gray-600" />`

Handles (per D-13 -- connection handles for edge creation):
- `<Handle type="source" position={Position.Right} className="!bg-indigo-500 !w-2.5 !h-2.5" />`
- `<Handle type="target" position={Position.Left} className="!bg-indigo-500 !w-2.5 !h-2.5" />`

State: `const [showPopover, setShowPopover] = useState(false)` -- toggled by pencil click.

When `showPopover` is true, render `<EditPopover>` adjacent to the node. The popover receives: `nodeId={id}`, `urlTemplate={data.urlTemplate}`, `pageCount={data.pageCount}`, `onSave={(urlTemplate, pageCount) => { ... }}`, `onClose={() => setShowPopover(false)}`.

The `onSave` must update the node data. Since custom nodes don't directly get `setNodes`, use a React context or pass an update callback through node data. The cleanest approach for React Flow v11: pass an `onUpdate` function in node data from App.tsx. So node data becomes `{ urlTemplate, pageCount, onUpdate }` where `onUpdate: (id: string, data: Partial<UrlNodeData>) => void`.

Export: `export const UrlNode = memo(UrlNodeComponent)` (memo for React Flow performance).

Structure:

- Two fields stacked vertically with `space-y-4` gap.

Field 1 -- URL Template:
- Label: `<label className="block text-xs font-semibold text-gray-700 mb-1">URL Template</label>`
- Input: `<input type="text" className="w-full h-9 text-sm text-gray-900 border border-gray-300 rounded-md px-3 focus:outline-none focus:ring-2 focus:ring-indigo-500" placeholder="/page/<id>" />`

Field 2 -- Page Count:
- Label: `<label className="block text-xs font-semibold text-gray-700 mb-1">Page Count</label>`
- Input: `<input type="number" min={1} className="w-full h-9 text-sm text-gray-900 border border-gray-300 rounded-md px-3 focus:outline-none focus:ring-2 focus:ring-indigo-500" placeholder="1" />`

Local state: `const [localTemplate, setLocalTemplate] = useState(urlTemplate)` and `const [localCount, setLocalCount] = useState(pageCount)`.
Validation state: `const [error, setError] = useState('')`.

Dismiss behavior (per D-07 and UI-SPEC Validation):
- Click outside: use a `useEffect` with `mousedown` listener on `document`. On outside click, if `localTemplate.trim() === ''`, set error to `"URL template cannot be empty"` and keep popover open. Otherwise call `onSave(localTemplate.trim(), Math.max(1, localCount))` then `onClose()`.
- ESC key: `useEffect` with `keydown` listener. On Escape, call `onClose()` without saving (discard).
- Use a ref on the popover container div to detect "outside" clicks.
- Stop propagation on the popover div to prevent React Flow from interpreting clicks inside the popover as canvas interactions.

Error display: Below the URL Template input, conditionally render: `{error && <p className="text-xs text-red-500 mt-1">{error}</p>}`. Clear error onChange.

5. **Drag-to-add from sidebar (D-08, CANVAS-01):**
```tsx
const onDragOver = useCallback((event: React.DragEvent) => {
  event.preventDefault();
  event.dataTransfer.dropEffect = 'move';
}, []);

const onDrop = useCallback((event: React.DragEvent) => {
  event.preventDefault();
  const type = event.dataTransfer.getData('application/reactflow');
  if (type !== 'urlNode' || !reactFlowInstance) return;
  const position = reactFlowInstance.screenToFlowPosition({
    x: event.clientX,
    y: event.clientY,
  });
  const newNode = createNode(position);
  setNodes((nds) => nds.concat(newNode));
}, [reactFlowInstance, createNode, setNodes]);
```

const onAddNode = useCallback(() => {
