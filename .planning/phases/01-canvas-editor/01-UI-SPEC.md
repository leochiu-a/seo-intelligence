## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gap inside node card (pencil icon inset), inline badge padding |
| sm | 8px | Node internal padding (top/bottom), edge label padding |
| md | 16px | Node internal padding (left/right), sidebar item padding, popover padding |
| lg | 24px | Sidebar total left padding, section separation inside popover |
| xl | 32px | Sidebar panel width padding, toolbar height |
| 2xl | 48px | Sidebar panel min-width |
| 3xl | 64px | Node card min-width |

Exceptions:
- Left accent border on node card: 4px width (visual weight, not spacing)
- Node card min-height: 64px (ensures URL template + page count are legible)
- Touch/click target for pencil icon: 32px × 32px minimum (hover target padding, not visual size)
- Toolbar height: 48px fixed

## Typography

| Role | Size | Weight | Line Height |
|------|------|--------|-------------|
| Body | 14px | 400 (regular) | 1.5 |
| Label | 12px | 600 (semibold) | 1.4 |
| Heading (node URL template) | 14px | 600 (semibold) | 1.3 |
| Secondary (page count, edge count) | 12px | 400 (regular) | 1.4 |

**Application rules:**
- Node URL template string (`/category/<dest>/list`): 14px / semibold / line-height 1.3 — primary label
- Node page count ("100 pages"): 12px / regular / line-height 1.4 — secondary info below template
- Sidebar item label: 14px / regular / line-height 1.5
- Popover field label ("URL Template", "Page Count"): 12px / semibold / line-height 1.4
- Toolbar button label ("+ Add Node"): 14px / semibold / line-height 1.5
- Edge link-count label: 12px / semibold / line-height 1.4 (must be legible on canvas background)

Font weights are exactly 2: **400 (regular)** and **600 (semibold)**. No other weights permitted in this phase.

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `#F9FAFB` (gray-50) | Canvas background — already set in App.tsx |
| Secondary (30%) | `#FFFFFF` (white) | Node card body, sidebar panel, popover background, toolbar background |
| Accent (10%) | `#6366F1` (indigo-500) | Node left accent border (4px), active/selected node ring, primary toolbar button background |
| Destructive | `#EF4444` (red-500) | Delete confirmation text only — no other use |

**Accent reserved for:**
1. Node card left accent border (always-on, 4px solid indigo-500)
2. Selected node ring/outline (2px ring, indigo-500 at 50% opacity outer glow)
3. "+ Add Node" toolbar button background (indigo-500, white text)
4. Selected edge stroke color (indigo-500)

**Supporting palette (not primary but needed):**
- Node card border: `#E5E7EB` (gray-200) — subtle card outline
- Pencil icon on hover: `#6B7280` (gray-500)
- Sidebar panel border: `#E5E7EB` (gray-200) — right border separating sidebar from canvas
- Toolbar bottom border: `#E5E7EB` (gray-200)
- Popover shadow: Tailwind `shadow-lg` (system shadow token)
- Edge default stroke: `#9CA3AF` (gray-400)
- Edge label background: `#FFFFFF` (white pill behind the count number)
- Canvas grid dots: `#D1D5DB` (gray-300) — React Flow Background default
- Inline validation error text: `#EF4444` (red-500)

**Primary visual anchor:** the canvas area is the primary visual anchor; the indigo-500 left accent border on each node card is the focal color element that draws the eye.

Source: D-04 (accent border color = indigo), App.tsx (`bg-gray-50` canvas background).

## Node Component Contract

**UrlNode card dimensions and style:**
- Min-width: 200px, max-width: 280px
- Min-height: 64px
- Background: white
- Border-radius: 8px (rounded-lg)
- Border: 1px solid gray-200
- Left accent border: 4px solid indigo-500 (implemented as border-l-4)
- Box shadow: `shadow-sm` (default), `shadow-md` + indigo ring on selected
- Padding: 8px 16px (8px top/bottom, 16px left/right)

**Node content layout (top to bottom):**
1. URL template string — 14px / semibold / gray-900 — single line, truncate with ellipsis
2. Page count — 12px / regular / gray-500 — e.g. "100 pages"

**Pencil icon (hover state):**
- Position: absolute, top-right corner (top: 8px, right: 8px)
- Icon: `Pencil` from lucide-react, size 14px, color gray-400
- Accessible label: `aria-label="Edit node"` on the button wrapping the icon
- Visible only on node hover (opacity-0 default → opacity-100 on group-hover)
- Click target: 28px × 28px (4px padding around 14px icon → rounds to 28px, acceptable for desktop PM tool)
- Hover state on icon itself: gray-600

**Node states:**
- Default: white bg, gray-200 border, shadow-sm
- Hover: shadow-md, pencil icon visible
- Selected: shadow-md, 2px indigo-500 ring (outline-style ring via React Flow selected class)
- Dragging: shadow-lg, slight opacity 0.9

Source: D-01, D-02, D-03, D-04, D-05.

## Edit Popover Contract

**Trigger:** Click pencil icon on node hover (D-05).

**Dimensions:**
- Width: 280px fixed
- Padding: 16px all sides

**Style:**
- Background: white
- Border: 1px solid gray-200
- Border-radius: 8px (rounded-lg)
- Box shadow: shadow-lg
- z-index: above canvas nodes (z-50)

**Positioning:** Adjacent to node, preferring right side; flip to left if within 300px of viewport right edge. Never off-screen.

**Fields (D-06):**
1. "URL Template" — text input, full width, placeholder `/page/<id>`
2. "Page Count" — number input, full width, min=1, placeholder `1`

**Field style:**
- Label: 12px / semibold / gray-700, margin-bottom 4px
- Input: 14px / regular / gray-900, height 36px, border 1px gray-300, rounded-md, padding 8px 12px
- Focus state: indigo-500 ring (ring-2 ring-indigo-500)
- Field gap: 16px between fields

**Validation — URL Template field:**
- When the URL Template input is empty on dismiss (click outside or blur): show inline error message directly below the input field.
- Error message: "URL template cannot be empty"
- Error text style: 12px / regular / red-500, margin-top 4px
- The popover remains open until the field is non-empty (prevent save/close while invalid on outside-click dismiss; ESC still discards and closes regardless).

**Dismiss behavior (D-07):** Click outside → validate then save and close (blocked if URL Template is empty — see Validation above). ESC key → discard changes and close.

## Edge Component Contract

**LinkCountEdge style:**
- Stroke: gray-400 (default), indigo-500 (selected)
- Stroke-width: 2px
- Type: smoothstep (React Flow default bezier/smoothstep — Claude's discretion as per CONTEXT.md)
- Animated: false (static arrow)
- Arrow: React Flow default MarkerEnd arrowhead, gray-400 fill

**Edge label (D-11, D-12):**
- Content: link count number (e.g. "3")
- Background: white pill — rounded-full, px-8px py-4px, border 1px gray-200
- Font: 12px / semibold / gray-700
- Clickable region: entire pill
- Editing state: `<input type="number">` replaces label text, same pill container, width 48px, indigo-500 ring-2
- Confirm: Enter key or click outside (blur)
- Default link count for new edges: `1`

**Edge states:**
- Default: gray-400 stroke, white label pill
- Selected: indigo-500 stroke, white label pill with indigo-200 border
- Editing label: input visible, indigo-500 ring on pill

## Interaction Contracts

| Interaction | Trigger | Behavior |
|-------------|---------|----------|
| Add node (primary) | Drag "URL Node" card from left sidebar onto canvas | Node spawns at drop position with defaults: `/page/<id>`, page count 1 |
| Add node (secondary) | Click "+ Add Node" in toolbar | Node spawns at canvas center |
| Move node | Drag node body (not handle) | React Flow native drag; node repositions |
| Select node | Click node body | React Flow native selection; selected state applied |
| Multi-select nodes | Shift+click or drag selection box | React Flow native |
| Delete node | Select → Delete or Backspace key | React Flow `onNodesDelete`; edges connected to node also removed |
| Open edit popover | Click pencil icon (visible on hover) | Popover opens adjacent to node |
| Dismiss popover | Click outside or press ESC | Save (outside click, blocked if URL Template empty) or discard (ESC) |
| Create edge | Drag from node connection handle to another node | React Flow native `onConnect`; edge spawns with link count 1 |
| Select edge | Click edge line | React Flow native selection |
| Edit edge label | Click edge label pill | Inline number input appears; Enter/blur confirms |
| Delete edge | Select edge → Delete or Backspace key | React Flow `onEdgesDelete` |
| Pan canvas | Drag blank canvas area | React Flow native pan |
| Zoom canvas | Scroll wheel or pinch | React Flow native zoom |
