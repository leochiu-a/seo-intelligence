  <behavior>
    ## calculatePageRank(nodes, edges) -> Map&lt;string, number&gt;

    - Empty graph (no nodes): returns empty Map
    - Single node, no edges: returns Map { nodeId => 1.0 }
    - Two disconnected nodes: each gets score 1.0
    - Two nodes A->B (linkCount=1, pageCount=1 each): B gets higher score than A
    - Three-node chain A->B->C: C > B > A
    - Cycle A->B->A: both get equal scores (symmetry)
    - Link count weighting: A->B with linkCount=5 transfers more equity than linkCount=1
    - Page count weighting: node with pageCount=100 has a larger equity pool than pageCount=1
    - Convergence: scores sum to N (number of nodes) within floating point tolerance
    - Dampening factor d=0.85 applied correctly

    ## classifyScoreTier(score, allScores) -> ScoreTier

    - Single score array: returns 'neutral'
    - All equal scores: returns 'neutral'
    - Score in top third of range: returns 'high'
    - Score in middle third: returns 'mid'
    - Score in bottom third: returns 'low'

    ## identifyWeakNodes(scores) -> Set&lt;string&gt;

    - Empty map: returns empty Set
    - Single node: returns empty Set (no outliers possible)
    - All equal scores: returns empty Set (stddev = 0, no weak nodes)
    - Node below mean - 1 stddev: included in weak Set
    - Node above mean - 1 stddev: not included
    - Fallback: nodes with zero inbound (score at base dampening level) detected as weak when stddev is meaningful
  </behavior>

    export type ScoreTier = 'high' | 'mid' | 'low' | 'neutral';

    /**
     * Iterative PageRank with page count and link count weighting.
     * d = 0.85, convergence delta < 0.0001, max 100 iterations.
     *
     * Algorithm:
     * 1. Initialize all scores to 1.0
     * 2. For each iteration:
     *    a. For each node u, compute:
     *       PR(u) = (1 - d) + d * sum over all v linking to u of:
     *         PR(v) * linkCount(v->u) * pageCount(u) / totalWeightedOutbound(v)
     *       where totalWeightedOutbound(v) = sum of (linkCount(v->w) * pageCount(w)) for all w that v links to
     *    b. Check convergence: max absolute delta across all nodes < 0.0001
     * 3. Return Map<nodeId, score>
     */
    export function calculatePageRank(
      nodes: Node<UrlNodeData>[],
      edges: Edge<LinkCountEdgeData>[],
    ): Map<string, number> { ... }

     * Classifies a score into a tier based on relative thirds of the score range.
     * If only one unique score exists, returns 'neutral'.
     */

     * Identifies weak nodes: those with score below (mean - 1 stddev).
     * If stddev is 0 (all equal), returns empty set.
     */
    export function identifyWeakNodes(
      scores: Map<string, number>,
    ): Set<string> { ... }

    Implementation details for `calculatePageRank`:
    - Build adjacency: `Map<sourceId, Array<{ targetId, linkCount }>>` from edges
    - Build inbound: `Map<targetId, Array<{ sourceId, linkCount }>>` from edges
    - Compute `totalWeightedOutbound(v)` = sum of `edge.linkCount * targetNode.pageCount` for all outgoing edges of v
    - Iterate: `newScore[u] = (1 - 0.85) + 0.85 * sum(scores[v] * linkCount(v->u) * pageCount(u) / totalWeightedOutbound(v))`
    - Converge when `Math.max(...deltas) < 0.0001` or iterations >= 100
    - Return new Map with final scores

    Implementation details for `classifyScoreTier`:
    - Find min and max of allScores
    - If min === max (or allScores.length <= 1), return 'neutral'
    - Compute thresholds: `lowThreshold = min + range/3`, `highThreshold = min + 2*range/3`
    - score >= highThreshold -> 'high', score >= lowThreshold -> 'mid', else -> 'low'

    Implementation details for `identifyWeakNodes`:
    - If scores.size <= 1, return empty Set
    - Compute mean and stddev of score values
    - If stddev === 0 (use < 0.00001 for float safety), return empty Set
    - threshold = mean - stddev
    - Return Set of nodeIds where score < threshold
