# 999.6: Anchor Text Type on Edge (P1)

## 問題

Anchor text 是內部連結 SEO 的第二大變數：

- exact match / partial match / branded / generic 的比例影響排名
- 相同 link count，anchor 用「首頁」vs「台北一日遊」傳遞的 topical relevance 天差地遠
- 至少 edge 上要能標 anchor text type，讓使用者能看到「這個節點的 inbound anchor 是否過度單一」

## 目標

Edge 上可標 anchor text type（exact match / partial match / branded / generic），顯示節點的 inbound anchor 多樣性。

## 價值

補齊 SEO 第二變數。
