# 999.7: 匯入 Screaming Frog CSV / GSC Export (P1)

## 問題

這是 pre-deploy tool 最根本的信任問題 — PM 畫完圖、看到某頁分數是 0.47，那又怎樣？建議至少其中一個：

- 支援匯入 Screaming Frog / Sitebulb 的 crawl data 作為 baseline，算出「現況 PageRank」，對比 GSC 的 impressions / clicks ranking，證明模型和真實流量相關
- 或內建一組示範 scenario（例如一個常見電商架構），讓新用戶看得出分數高低和現實的對應關係

## 目標

支援匯入 Screaming Frog / Sitebulb crawl data 作為 baseline，對比 GSC impressions/clicks，證明模型和真實流量相關。

## 價值

讓模型和真實流量有橋樑，才敢拿去向老闆提案。
