       ```

       ```

    4. Edit `tailwind.config.js` to set content paths:
       ```js

         content: [
           "./index.html",
           "./src/**/*.{js,ts,jsx,tsx}",
         ],
         theme: {
           extend: {},
         },
         plugins: [],
       }
       ```

    5. Replace the contents of `src/index.css` with Tailwind directives:
       ```css
       @tailwind base;
       @tailwind components;
       @tailwind utilities;
       ```

    6. Delete `src/App.css` if it was created by the scaffold (styles will be in index.css and Tailwind utilities).

    7. Verify `vite.config.ts` contains the React plugin:
       ```ts
       import { defineConfig } from 'vite'
       import react from '@vitejs/plugin-react'

       export default defineConfig({
         plugins: [react()],
       })
       ```

    8. Ensure `src/main.tsx` imports `./index.css` and renders App:
       ```tsx

       ReactDOM.createRoot(document.getElementById('root')!).render(

         </React.StrictMode>,
       )
       ```
